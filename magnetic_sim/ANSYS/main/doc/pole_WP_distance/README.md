# doc/pole_WP_distance/ — 磁極尖端到工作點（WP）距離

**用途**：推導 hexapole 各磁極尖端到工作空間中心（WP）的絕對距離與其在 (x,y,z) 的分量 —— 以工作空間半徑 ℓ 表示，例如 (√2/2·ℓ, √6/6·ℓ, √2/2·ℓ)，供點電荷模型 kernel 與幾何對齊使用。
**內容**：LaTeX 原稿（.tex）+ 編譯 PDF，依 model topic 分群。代表主題：`pole_distance_hexapole`（六極尖端-WP 距離分量）。
**命名 / 慣例**：analysis-first schema：`<analysis=pole_WP_distance>/<topic=model 名>/{scripts(.tex), pdf}`；第二層 topic = 物理模型名（目前 `long2016_hexapole_halfcut`）。
**相關**：見 ../README.md、../../CLAUDE.md；ℓ 定義見 memory `terminology_l`；ANSYS-CAD 尺寸對齊見 .claude/rules/ansys-cad-alignment.md。
