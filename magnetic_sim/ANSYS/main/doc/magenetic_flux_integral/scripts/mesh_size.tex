\section*{1. 網格數量與網格間距}

網格數量:
\[
N = 81 \times 81
\]

網格間距:
\[
\begin{cases}
\Delta y = \dfrac{2R}{80} = \dfrac{2 \cdot 40\,\mu\mathrm{m}}{80} = 1\,\mu\mathrm{m} \\[10pt]
\Delta z = \dfrac{R}{80} = \dfrac{40\,\mu\mathrm{m}}{80} = 0.5\,\mu\mathrm{m}
\end{cases}
\]

網格面積元素:
\[
\Delta A = \Delta y \cdot \Delta z = 0.5\,\mu\mathrm{m}^2
\]
