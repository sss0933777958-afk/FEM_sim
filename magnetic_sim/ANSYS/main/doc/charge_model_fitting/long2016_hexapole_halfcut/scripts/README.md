# doc/charge_model_fitting/long2016_hexapole_halfcut/scripts/ — 半切 hexapole 擬合原稿 + 產生器

**用途**：存放半切 hexapole 點電荷擬合的 **LaTeX 原稿（.tex）** 與 **LaTeX 表格產生器（.m）**。
**內容**：`fit_error_def.tex`（誤差定義向量圖，仿 Long 2016 Fig 2.6c）；`gen_KI_latex.m`、`gen_KI_ball_latex.m`、`gen_KI_R040_latex.m`、`gen_fit_J_cube40_latex.m`（各取樣範圍的 K̂_I / 擬合結果 .tex 表格產生器）。
**命名 / 慣例**：葉夾同時放 .tex 原稿與 .m 產生器（本 topic 無對應 pdf/ 葉夾）；`gen_*_latex.m` 讀 fit 結果輸出 .tex；後綴 `ball`/`R040`/`cube40` 標取樣範圍。
**相關**：見 ../README.md、../../../CLAUDE.md。
