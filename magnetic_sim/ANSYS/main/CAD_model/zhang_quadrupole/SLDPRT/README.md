# CAD_model/zhang_quadrupole/SLDPRT/ — Zhang Quadrupole SolidWorks 原檔

**用途**：Zhang 4-pole（0/90/180/270° lozenge + yoke + pillars）Quadrupole 的 SolidWorks 可編輯母檔。

**內容**：主線 `0.46mm_baseline.SLDPRT` + revision `0.46mm_baseline_V2.SLDPRT`…`_V10.SLDPRT`；變體 `0.46mm_20um.SLDPRT`、`0.67mm_baseline.SLDPRT`；原始件 `zhipeng.SLDPRT`。

**資料來源 / 流向**：母檔 → STEP 匯出（`../STEP/` 目前為空，需要時再產）→ IGES pipeline。

**命名 / 慣例**：`0.46mm` / `0.67mm` = 極尖距變體，`_V*` = 設計修訂，`_20um` = tip 圓角變體；`zhipeng` 為原作者原始件名。
