# CAD_model/long_fei/SLDPRT/ — Long Fei hexapole SolidWorks 原檔

**用途**：Long Fei 六極半切 hexapole 的 SolidWorks 可編輯母檔（上/下磁極 + sensor 位置）。

**內容**：`Bottom pole.SLDPRT`（下極半切件）、`Magnetic pole Bottom_sensor_location.SLDPRT`、`upper_pole_sensor_location.SLDPRT`（標註 Hall sensor 放置位置的上/下極）。

**資料來源 / 流向**：母檔 → `../STEP/` 匯出 → IGES pipeline。

**命名 / 慣例**：`.SLDPRT` 元件母檔；topic 名 `long_fei` = 下游 `long2016_hexapole_halfcut`（同物理模型，命名不一致）。
